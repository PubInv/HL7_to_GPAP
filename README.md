# HL7_to_GPAP
HL7 to GPAP bidirectional adapter

This project is a meant to be a Python project that produces an easy-to-use bidirectional adapter 
between the HL7 protocol and the Public Invention [General Purpose Alarm Protocol](https://github.com/PubInv/krake) (GPAP).

HL7 is a widely used health protocol for many purposes, including alarms.  
The GPAP is a new protocol designed specifically to work with the [ADaM](https://github.com/PubInv/ADaM) and Krake projcts.
Those three projects are Public Invention's attempt to empower open-source 
humanitarain enginers to build safe and effective alarm systems for medical devices,
hobby projects, research projects, and pretty much anything else.

# Use Case: A Clinical Alarm Management System

A significant body of research exists on alarm fatigue and alarm management. 
Some firms make complete alarm management systems. 
We would like to democratize and open this to allow many different devices 
to work together in a harmonious system that allows open-source medical 
devices to be effectively used.

# Use Case: Monitoring a Trading Market

We can imagine that an alarm when the price of a commodity or equity rises 
or falls at threshold or very quickly and very slowly.
